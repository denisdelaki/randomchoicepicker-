# The choicePicker

#### By **KIPYEGON DENIS**

## Description

the choicepicker webpage is used to make decision where one enters the choices in the text area and allow the site to randomly pick the choice for him/her. the choicepicking is completely random

## Setup/Installation Requirements

- Visual studio
- browser-chrome

## Known Bugs

the page does not allow one to set the interval while picking the choices

## Technologies Used

Javascript HTML and CSS
